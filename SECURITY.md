# Security Policy

## Supported Versions

We support security updates for the following versions of the kit:

| Version | Supported |
| ------- | --------- |
| < 0.1.x | Yes |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please report it immediately by opening a private report or contacting the maintainers directly.

We appreciate your help in keeping the CoreZero framework secure!
